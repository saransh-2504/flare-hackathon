Strategies

POST /api/strategies

Create a new automation strategy

Parameters:

Parameter Type Description
triggerType string price, event, or time
asset string Asset symbol or event name
condition string Trigger condition
action string mint, swap, transfer, redeem
amount number Amount in USD
protected boolean Enable security firewall
GET /api/strategies

List all your strategies

Security

GET /api/security/status

Get current security status and threat level

Response:

{
  "success": true,
  "data": {
    "threatLevel": "SAFE",

    "threatsDetected": 0,

FTSO Price Feeds

GET /api/ftso/price/:symbol

Get current price for an asset

Example:

GET /api/ftso/price/BTC

Response:

{
  "success": true,
  "data": {
    "symbol": "BTC",
    "price": 50000,
    "decimals": 18,
    "timestamp": "2024-12-07T10:30:00Z"
  }
}
