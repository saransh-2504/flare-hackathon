Authentication

  "name": "Your Name"

{
  "success": true,
  "apiKey": "flr_abc123...",
  "tier": "free",
  "rateLimit": 100
}

Strategies

POST /api/strategies

Create a new automation strategy

Parameters:

Parameter Type Description
triggerType string price, event, or time
asset string Asset symbol or event name
condition string Trigger condition
action string mint, swap, transfer, redeem
amount number Amount in USD
protected boolean Enable security firewall
GET /api/strategies

List all your strategies
