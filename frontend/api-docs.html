Authentication

POST /api/auth/register

Register and receive your API key

Request Body:

  "email": "your@email.com",
  "walletAddress": "0x...",
  "name": "Your Name"
}

Response:

{
  "success": true,
  "apiKey": "flr_abc123...",
  "tier": "free",
  "rateLimit": 100
}

Strategies

POST /api/strategies

Create a new automation strategy

Parameters:

Parameter Type Description
triggerType string price, event, or time
asset string Asset symbol or event name
condition string Trigger condition
action string mint, swap, transfer, redeem
amount number Amount in USD
protected boolean Enable security firewall
GET /api/strategies

List all your strategies

Security

GET /api/security/status

Get current security status and threat level

Response:

{
  "success": true,
  "data": {
    "threatLevel": "SAFE",
    "circuitBreakerActive": false,
    "threatsDetected": 0,
    "sources": {
      "certik": { "status": "operational" },
      "peckshield": { "status": "operational" }
    }
  }
}

FTSO Price Feeds

GET /api/ftso/price/:symbol

Get current price for an asset

Example:

GET /api/ftso/price/BTC

Response:

{
  "success": true,
  "data": {
    "symbol": "BTC",
    "price": 50000,
    "decimals": 18,
    "timestamp": "2024-12-07T10:30:00Z"
  }
}
